Firebase setup (one-time, ~5 min)
Follow these steps to get your free database, then paste your config below.
1
Go to console.firebase.google.com and sign in with your Google account
2
Click Add project, name it "couple-budget", skip Google Analytics, click Create project
3
Click the web icon </>, register app as "budget-app", click Register app
4
Copy the firebaseConfig object values and paste them below
5
In the Firebase console go to Build → Firestore Database → Create database → Start in test mode → Enable
